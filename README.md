# Jogo da Trilha

Projeto de jogo da trilha que utiliza comunicação por sockets, com chat para iteração. 


## Instalação

Baixe.
"Jogo da Trilha.exe"
Coloque um IP caso já tenham criado um servidor, se não crie um servidor.
Coloque um Nick.
Jogue. :)
